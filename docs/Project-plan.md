## Project Overview

**Client:** Barbara Bates, Nottingham Growing Network
**Contact:** barbara@bulwellforestgarden.co.uk

NGN is a newly constituted umbrella network supporting 25+ community food growing spaces across Nottingham. The team was tasked with building a professional communications system and public-facing website.

## Priorities (in order)

1. Communication and document storage system for trustees
2. Public-facing website with members area
3. Digital map of all 35 community gardens
