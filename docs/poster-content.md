# Poster Content Summary
## Group 2 - Nottingham Growing Network
## ISYS20182 PPM - Nottingham Trent University 2025/26

---

## Problem

NGN had no professional online presence, no internal communication system
for trustees, and no way for the public to find their local community garden.

Three key problems:
- No professional trustee email addresses or shared document storage
- No central public resource to find NGN gardens and events
- Unmanaged social media with no coherent strategy

---

## Aims & Objectives

Aim: To design and develop a professional communications system and
public-facing website for NGN enabling trustee collaboration and helping
the public find their nearest community garden.

Objectives:
- Set up free Gmail communication system for 5 trustees
- Create shared Google Drive for trustee document collaboration
- Build a public-facing website with 7 fully functional pages
- Develop an interactive map showing all 35 garden locations
- Create a password-protected members area for trustees
- Produce a social media improvement plan

---

## Our Solution

### 1. Public Website
- Built using HTML, CSS and JavaScript hosted on GitHub Pages (free)
- 7 pages: Home, About, Gardens, Events, Recipes, Members, Get Involved
- Interactive Google Maps embed with all 35 gardens
- Searchable garden directory
- Real photos from Barbara throughout
- Contact form, FAQ and newsletter signup
- Live at: t0332876-hub.github.io/Group-2-NGN/

### 2. Trustee Communication System
- Researched 3 options: free Gmail, Workspace Nonprofit, paid plan
- Recommended two-phase approach
- Created test Gmail account as proof of implementation
- Set up shared Google Drive with 5 subfolders
- Password-protected members area with noticeboard, calendar and directory

### 3. Social Media Plan
- Audited existing Facebook and Instagram
- Recommended Instagram as main controlled channel
- Proposed weekly posting schedule of minimum 3 posts per week

---

## Technologies Used

- HTML5, CSS3, JavaScript
- GitHub Pages (free hosting)
- Google My Maps (35 garden pins)
- Google Drive and Gmail
- MS Planner (project management)
- GitHub (version control)

---

## Key Challenges

- Google blocked multiple Gmail creation from one device
  Solution: recommended each trustee creates their own account
- Client did not respond to Gmail options email
  Solution: documented research and implementation attempt as evidence
- Image file names with spaces broke the website
  Solution: URL-encoded all file names in HTML code
- Initial website wording was inaccurate
  Solution: rewrote all content after client feedback

---

## Conclusions

The project successfully delivered a live professional website for NGN
at no cost. All three client priorities were addressed. Client feedback
was incorporated across two rounds of improvements.

## Limitations

- Gmail setup not fully completed due to no client response
- Contact form does not send real emails (requires backend server)
- Members password stored in browser local storage only
